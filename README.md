# Malla Interactiva LICC UC
Pasaste algún ramo? Te echaste algún ramo? Aquí puedes ver todo el camino que queda, puedes mover el ramo que te echaste para otro semestre y ver como quedaría todo! Resuelve como si fuera Tetris!

IMPORTANTE!
Recuerda que la malla puede estar sujeta a cambios o no este tan exactas las cadenas, si encuentras algun fallo avisar para modificar :p
